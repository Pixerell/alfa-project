# Небольшой проект с использованием React/Redux/Axios, всё с Typescript

.Проект использует апишку фейкового магазина с этого сайта - https://fakestoreapi.com/
.Проект разбит на 4 главные папки: store с редаксом, api, domain с логикой и ui
.Внутри проекта реализована фильтрация (по названию ><, цене >< и понравившемся продуктам), поиск по имени, пагинация по 10 продуктов, добавление продуктов через отдельную страницу с превьюшкой редакции в реальном времени
.Создан с помощью Vite. Линтер esLint и Prettier
